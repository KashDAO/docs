Shell tab-completion utilities.

> Generated from `kash docs --json`. Each command's behaviour is documented in-binary via `kash completion <subcommand> --help` — this page is the structured reference.

## Usage

```bash
kash completion <subcommand> [options]
```

## Subcommands

### `kash completion install`

Install bash/zsh/fish completion to your shell config.

### `kash completion uninstall`

Remove kash completions from your shell config.
